# 💎 CoreVault AI: Institutional-Grade Wealth Guardian
### 🏆 Built for Professional Asset Management & Risk Diagnostics

**CoreVault AI** is a high-performance wealth management platform that bridges the gap between retail investing and institutional risk management. Featuring real-time AI "Sentinel" diagnostics, automated target order execution, and deep "Investor DNA" profiling, it provides a professional-grade command center for the modern investor.

---

## 🚀 The Vision
In the era of extreme market volatility, traditional dashboards are too static. Retail investors often lack the sub-second risk assessment tools used by hedge funds. **CoreVault AI** is designed to be your "Digital Sentinel"—a localized intelligence core that monitors your portfolio health 24/7 and triggers rescue alerts the moment a market anomaly is detected.

---

## ✨ Key Features

### 🛡️ 1. Market Sentinel AI (Real-Time Diagnostics)
- **Instant Risk Triage**: Uses a localized Gemini/Groq-powered core to monitor holdings for severe slippage or news-driven collapses.
- **Emergency Exit Alerts**: Triggers a high-priority "Liquidate" overlay during market crashes to preserve user capital.
- **Sentiment Mapping**: Integrated Fear & Greed indexing based on live volatility pulse.

### 🤖 2. "Hired Hand" Target Orders
- **Precision Execution**: Set automated BUY/SELL targets for Stocks, Crypto, and Mutual Funds.
- **Slippage Protection**: Algorithmic orders that execute exactly when the market hits your price point.
- **Universal Asset Support**: Unified interface for tracking traditional equity alongside high-volatility digital assets.

### 🎓 3. CoreVault Academy (Skill Verification)
- **Module Learning**: Interactive knowledge modules covering everything from options flow to asset allocation.
- **Daily Verification Challenge**: A randomized 5-question MCQ challenge that awards XP, Badges, and Streaks.
- **Institutional Certification**: Track your "Investor Rank" based on proven performance and knowledge scores.

### 🪪 4. Secure "Investor DNA" Identity
- **Decentralized Profile**: Secure, offline-first biometric/data storage using local persistence.
- **Scannable ID Pass**: Generates an optimized, scannable QR pass containing your encrypted Investor DNA (Risk profile, goals, ID).
- **Cross-Platform Readiness**: Designed to be portable across the CoreVault network in a single scan.

## 🛠️ Technology Stack
- **Frontend**: Next.js 14, TypeScript
- **State Management**: React Hooks (useAuth, useSentinel) & Universal Pulse Engine.
- **Styling**: Vanilla CSS + Tailwind utility layer for institutional glassmorphism.
- **Animations**: Framer Motion (Smooth page transitions & 🚨 alert pulses).
- **Charts**: Recharts (Mark-to-market performance audit).
- **Identity**: `qrcode.react` with custom SVG data injection for scannable passes.

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/corevault-ai.git
cd corevault-ai
```

### 2. Install dependencies
```bash
npm install
```

### 3. Launch Development Server
```bash
npm run dev
```
